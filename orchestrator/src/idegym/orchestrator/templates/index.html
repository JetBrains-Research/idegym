About

The orchestrator exposes a FastAPI-based control plane for launching and supervising ephemeral development environments. It tracks resource usage per client via configurable rules and keeps metadata about active servers.

Useful endpoints:

  • GET /health — orchestrator health check
  • GET /metrics — Prometheus metrics
